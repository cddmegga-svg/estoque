# Memória do Projeto

Esta pasta contém o histórico de planejamento e evolução do sistema, gerado pelo assistente de IA.

## Arquivos Principais

*   **`task.md`**: Lista de tarefas macro do projeto (To-Do List). Mostra o que já foi feito (Fases 1 a 15+) e o que está planejado.
*   **`walkthrough.md`**: Guia de funcionalidades entregues. Explica como usar as novas features (ex: Importação XML, Estoque Mínimo, Transferências).
*   **`implementation_plan.md`**: Detalhes técnicos da última fase implementada. Explica as mudanças no banco de dados, API e frontend.

## Como usar
Mantenha esses arquivos versionados no Git para que, ao trocar de PC ou desenvolvedor, o contexto do projeto não se perca.
